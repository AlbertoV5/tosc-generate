Understanding .tosc filesÂ¶

The files are formatted as XML trees with the declaration:

<?xml version='1.0' encoding='UTF-8'?>

And the root:

<lexml version='3'>

After the root, the main element is the node, which has this basic structure:

<root>
|__<node>
   |__<properties>
   |  |__<property>
   |     |__<key>
   |     |  |__key.text
   |     |__<value>
   |        |__value.text
   |        |__<x>
   |           |__x.text
   |__<values>
   |__<children>
     |__<node>
